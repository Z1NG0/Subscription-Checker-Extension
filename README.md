## Что делает расширение
- Добавляет кнопку `Проверить` на странице друзей (если выключена автопроверка)
- Определяет статус каждого друга: `PREMIUM`, `LITE` или `NONE`
- Показывает сводную статистику (`PREMIUM | LITE | None`) в панели `[CS-SUB]`
- Поддерживает автопроверку при переходе на страницу друзей

## Настройки (popup)
Нажмите на иконку расширения и настройте:

1. `Показывать меню [CS-SUB]`
- Включено: панель со статусом и счетчиками отображается
- Выключено: панель скрыта

2. `Визуальная проверка`
- Включено: во время проверки показывается промежуточная подсветка карточек
- Выключено: итоговый статус применяется сразу (быстрее)

3. `Автопроверка друзей`
- Включено: проверка запускается автоматически при входе на страницу друзей, кнопка `Проверить` скрывается
- Выключено: запуск проверки вручную через кнопку `Проверить`

## Установка
1. Скачайте или клонируйте папку проекта
2. Откройте `chrome://extensions`
3. Включите `Режим разработчика`
4. Нажмите `Загрузить распакованное расширение`
5. Выберите папку проекта `Subscription-Checker-Extension`

## Как использовать
1. Откройте профиль на `cybershoke.net` и перейдите в раздел друзей (`/friends`)
2. Если автопроверка выключена, нажмите кнопку `Проверить`
3. Дождитесь окончания проверки и смотрите подсветку/счетчики

## Структура проекта
- `manifest.json` — манифест расширения (MV3)
- `content.js` — основная логика на странице
- `popup.html` — интерфейс настроек
- `popup.js` — сохранение/загрузка настроек (`chrome.storage.local`)
- `icons/` — иконки расширения
